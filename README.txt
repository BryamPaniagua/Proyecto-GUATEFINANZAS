El banco Guate-Finanzas está interesado en contratarnos para el desarrollo de la banca
virtual de usuario y administrador de cuentas bancarias.

El proyecto deberá ser realizado en C# utilizando MVC 5 de ASP.NET con Visual Studio 2013 y SQL
Server 2012, utilizando bitbucket.

Deberá realizar el correspondiente análisis, utilizando la especificación de requerimientos de
software, diagrama de clases, diagrama Entidad-Relación, Diagrama de casos de Uso, Diagrama de
Transición de Estados y Diagrama de Flujo de Datos para el sistema a realizar.
